2. Quiz Game (Full-stack)

This portfolio project showcases a comprehensive application that seamlessly integrates a full-stack web development setup, combining both front-end and back-end technologies. On the back-end, the Django framework manages database interactions with models like Registermodel, AddressDetails, and VendorDtails, ensuring efficient data storage and retrieval with CRUD operations exposed via RESTful APIs. This project demonstrates a robust understanding of web technologies and a strong foundation in developing scalable web applications.
